# Street Art Mobile Application (Android)

Name: Dale Healy Egan (20066529)

## Overview.

Street Art is a free downloadable android application built using kotlin that allows users to store the location of their favourite street art within a depository.

Users can add a description, artist name, image and placemark of the street art using google maps.


## Features

Add, List Delete and Update

Storage of User and Placemark using JSON

Basic authentication

Splash screen

Map usage
